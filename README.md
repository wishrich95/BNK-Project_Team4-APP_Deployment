# 🏦 금융권 예금상품 관리 시스템 2차 Project

## 딸깍은행 APP 개발

### 📌 프로젝트 개요

| 구분 | 내용 |
|------|------|
| **프로젝트 구분** | 금융·은행권 APP 서비스 개발 |
| **프로젝트 명** | 금융권 예금상품 통합 관리 시스템 - 부제: 딸깍은행 APP 개발 |
| **배경 및 목적** | <ul><li>금융권의 예금 및 금융상품 정보를 통합 관리할 수 있는 APP 서비스 구현</li><li>실제 은행 서비스 구조를 반영하여 금융 데이터 설계, 관리, 보안 흐름을 경험</li><li>기획부터 UI*UX, 프론트에서 백엔드까지 전 과정을 이해하고 협업 역량을 강화</li></ul> |
| **기대 효과** | <ul><li>금융권 APP 시스템 개발 프로세스 실무 감각 향상</li><li>UI/UX 설계와 관리자 페이지 구축 경험을 통한 실전 대응 능력 강화</li><li>데이터 중심의 시스템 관리 및 통계 기능 구현 경험 축적</li></ul> |
| **주요 기능** | <ul><li><b>사용자 화면:</b> 가입하기·로그인·로그아웃 / 예금상품 조회 / 상품상세 / 금융감정분석 / 번역(다국어)</li><li><b>관리자 화면:</b> 회원관리 / 상품관리 / 게시판관리 / 고객센터관리 / 환경설정</li><li><b>시각화 기능:</b> 판매통계 / 조회통계 / 상품유형 비율 원형그래프 / 감정분석 리포트</li></ul> |
| **서비스 채널** | 모바일 APP, PC Web (반응형) |
| **개발 환경** | <ul><li><b>Frontend:</b> HTML, CSS, JavaScript, Adobe XD</li><li><b>Backend:</b> Java 17, Spring Framework(Spring Boot 3.x), MyBatis</li><li><b>Database:</b> Oracle 21c XE</li><li><b>Server:</b> GCP EC2 / Tomcat 10</li><li><b>협업/CI:</b> GitHub / Figma / Notion</li></ul> |
| **작업 기간** | <ul><li>2025.12.09 (화) ~ 2026.01.07 (수) / 160h</li></ul> |
