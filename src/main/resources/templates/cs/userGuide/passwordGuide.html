비밀번호 안내

  • 딸깍은행 홈페이지에서 인터넷뱅킹을 하실 때 필요한 비밀번호에 대한 안내입니다.
  • 각 비밀번호는 서비스마다 다르게 쓰이고 있으므로 서비스 이용시에 유의하시기 바랍니다.

인증서암호

  • 인터넷뱅킹을 이용하기 위해서는 인증서를 발급받고 인증서를 제출(확인)해야 하는데 이때 사용되는 비밀번호입니다.
  • 인증서암호는 인증서를 발급받을 때 고객님께서 직접 만들어야 하며, 숫자·영문자·특수문자를 포함하여 10자리 이상으로 구성하여야 합니다.
Case 1
인증서암호를 변경하시고자 할 경우

공동인증센터 → 공동인증서관리 → 공동인증서(구 공인인증서) 선택 → 암호변경

암호변경
Case 2
인증서암호를 모르는 경우

공동인증센터 → 공동인증서발급(재발급)

인증서재발급

계좌비밀번호

  • 계좌비밀번호는 계좌(통장)의 숫자 4자리 비밀번호로, 영업점 창구와 인터넷뱅킹을 포함한 은행 거래 시마다 기본적으로 요구됩니다.
  • 인터넷뱅킹의 각종 조회, 송금 등 계좌와 관련된 거래 시 계좌비밀번호 입력이 필요합니다.
  • 계좌비밀번호를 모르는 경우나 3회 이상 틀린 경우 등 보안 사유에는 영업점을 직접 방문하여 조치를 받으셔야 합니다.

보안카드비밀번호

  • 보안카드비밀번호는 인터넷뱅킹 거래의 보안을 더욱 강화하기 위해 사용되는 비밀번호입니다.
  • 공동인증서(구 공인인증서), 송금/공유 등 거래에서 사용되며, 보안카드 코드번호와 함께 숫자 4자리를 입력하셔야 합니다.
  • 보안카드비밀번호를 3회 이상 틀린 경우 또는 보안카드를 분실한 경우에는 영업점을 직접 방문하셔야 합니다.
